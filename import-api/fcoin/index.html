如何导入Fcoin交易所的API
import-api

1、首先登陆fcoin交易所

2、点击右上角“账户–设置”，如下图

3、进入设置页面选择“API 设置”

4、在“CREATE API”页面创建自己的API，例如使用“test”作为api名称（方便记录），然后点击“创建新密钥”

5、按照提示输入谷歌验证码

6、记录Api key和Secret key

7、将生成的api key拷贝到app端，确认无误后点击导入
