如何导入币夫（Bitforex）交易所的API
import-api

1、首先登陆bitforex交易所

2、在页面右上角选择“资产管理–我的资产”，如下图

3、进入页面之后，在页面左侧选择“API管理”

4、创建API key

点击“创建API”（如果已有，可直接查看已有的api aky），例如使用bitower作为名称生成api，权限只有“交易”，绑定IP可以从www.ip138.com中获取（请在常用环境中获取），然后选择“确认”

5、输入短信验证码

6、记录Api key和Secret key

7、将生成的api key拷贝到

也可以通过app端导入页面的扫描二维码功能直接扫描上面页面的二维码
